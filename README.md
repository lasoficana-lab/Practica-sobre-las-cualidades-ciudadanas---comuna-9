# Practica-sobre-las-cualidades-ciudadanas- comuna-9
El presente repositorio reúne y organiza los documentos, evidencias y producciones desarrolladas durante la práctica profesional, llevada a cabo en la ciudad de Neiva, específicamente en la Comuna 9.
Tiene como objetivo, organizar y evidenciar el proceso desarrollado durante la práctica profesional, destacando el fortalecimiento de las cualidades ciudadanas en estudiantes de grado tercero.

Cualidades ciudadanas trabajadas:
•	Autonomía
•	Diálogo
•	Deliberación
•	Participación

 Estructura del repositorio
•	Planeaciones de clase
•	Talleres e intervenciones
•	Bitácoras reflexivas
•	Informes
